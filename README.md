# Herbal Wrap

Developing of web instrument

### We use

[react.js](http://facebook.github.io/react/index.html)
[material-ui](http://www.material-ui.com/)
[react-router](https://github.com/rackt/react-router)

### Run

```sh
$ npm install -g jspm
$ npm install -g gulp
$ npm install
$ jspm install
$ gulp
```
